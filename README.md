# Portal Segurança Digital

Portal informativo de utilidade pública para prevenção de golpes e acesso a informações de segurança pública.

O projeto utiliza tecnologias de desenvolvimento web para criar uma plataforma digital voltada á disseminação de infromações  públicas e segurança digital.

## Requesitos:
- python
- pip (gerenciador de pacotes do python)


## Como rodar o projeto(Windows)

É recomomendado usar um ambiente virtual

Na raiz do projeto execute o comando: 
```bash
python -m venv venv
```

Depois ative com:

```bash
venv/Scripts/active
```
por fim: 

```bash
pip install -r requeriments.txt
```
